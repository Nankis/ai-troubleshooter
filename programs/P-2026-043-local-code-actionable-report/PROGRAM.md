# P-2026-043 Local Code Actionable Report

## 背景

用户指出当前本地代码辅助排查只返回路径、命中词和行号，开发者看完仍不知道应该去哪个类/方法、哪几行、怀疑什么问题，也不知道下一步怎么核对，因此功能价值不足。

## 目标

- 本地 debug-only 代码排查输出可操作定位卡片。
- 每个命中至少尽量给出：相对文件、语言、具体类/方法/函数、命中行、建议查看行范围、可疑原因、下一步核对建议。
- 在本地 allowlist 仓库内允许返回有限、脱敏、短行的代码摘录，帮助开发者快速定位；生产路径仍禁止自动读业务仓库。
- Web Chat 回复改为清晰的多行开发者报告，不再把命中列表拼成一行。
- 更新测试和安全/运行说明。

## 非目标

- 本轮不实现完整 LSP/LSIF 语义引擎。
- 本轮不自动修改业务代码。
- 本轮不把本地代码线索升级为生产证据。
