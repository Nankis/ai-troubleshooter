# P-2026-057 Repo Root Edit Discipline

## 背景

桌面线程默认 cwd 可能与当前任务目标仓库不同。近期 `apply_patch` 曾把本应写入 `ai-troubleshooter` 的文件落到相邻项目 `AI超级个体`，虽然已迁回并清理，但这是重复出现的操作纪律问题。

## 目标

把“写入前确认目标仓库根目录”和“`apply_patch` 使用目标仓库绝对路径”固化为仓库级硬规则，避免再次污染其它项目。

## 范围

- 更新 `AGENTS.md`。
- 更新 `docs/LESSONS.md` 错误计数和复盘。
- 记录本 Program 的证据、交接和结果。

## 非目标

- 不修改业务代码。
- 不清理其它项目中不属于本次操作的文件或用户改动。
