# P-2026-042 Web Case Status Indicators

## 背景

用户指出左侧问题列表需要像 Codex 一样展示任务状态：AI 正在排查时显示转圈；AI 已经产出结果但用户还没查看时，要给出醒目的待查看状态。

## 目标

- 左侧 case 列表根据真实 case status 展示排查中状态。
- 对已产出结果且本机未打开查看的 case 展示待查看状态。
- 点击 case 后将该 case 最新结果标记为已查看。
- 保持状态含义清晰：不把前端已读状态写成后端业务状态。
- 实际启动 Web 页面验证。

## 非目标

- 本轮不新增服务端多用户已读表。
- 本轮不引入前端构建链路。
- 本轮不改变 case 后端状态机。
