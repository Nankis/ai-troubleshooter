# P-2026-018 Web Workbench Progress UI

## 背景

用户希望本地服务启动后有一个接近 Codex 的 Web UI，不只是聊天框，而是能看到问题排查进度、当前是否正在排查、右侧决策层步骤、左侧已注册 tools 和平台经验沉淀。

## 目标

- 将内置 Web Chat 升级为三栏排障工作台。
- 支持异步排查和状态轮询，避免请求阻塞期间看不到进度。
- 左侧展示 recent cases、Gateway tools、知识沉淀。
- 支持手动录入经验和软删除指定经验。
- 右侧从 AI decision logs 构建进度步骤。
- 用浏览器实际启动验证。

## 非目标

- 本轮不做用户登录和多租户权限。
- 本轮不引入前端构建链路，继续保持纯 HTML/CSS/JS 内置页面。
- 本轮不做 WebSocket；先用轮询保证简单可部署。
