# P-2026-039 业务方从零接入文档

## 背景

用户要求补齐一份面向完全不了解项目的业务接入方和业务方 AI 的接入文档，必须讲清楚：

- 业务方应该怎么用。
- LLM 在哪里配置、怎么配置。
- 需要建哪些平台表、怎么执行 migration。
- 运行哪些服务、各服务职责是什么。
- 下游怎么写只读接口。
- 写完接口后怎么注册到 Gateway，让平台实际用起来。

## 目标

把 `docs/business-onboarding-quickstart.md` 升级成可照着执行的从零接入手册，并同步 README 文档地图。

## 非目标

- 不改动代码实现。
- 不新增 DDL。
- 不做真实服务联调。
